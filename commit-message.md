# commit

### feat

实现功能或特性。

```
feat(模块): 添加了个很棒的功能
```

### fix

修复bug，模块可以为bug号，如fix(#001)。

```
fix(模块): 修复了一些 bug
```

### docs

更新文档相关的内容。

```
docs(模块): 更新了一下文档
```

### UI

更新UI样式相关的内容。

```
UI(模块): 修改了一下样式
```

### chore

对脚手架或配置项做的一些操作。

```
chore(模块): 对脚手架做了些更改
```

### locale

多语言相关内容。

```
locale(模块): 为国际化做了微小的贡献
```

### refactor

未改变任何功能，单纯的代码重构。

```
refactor(模块): 重构了locale组织形式
```
