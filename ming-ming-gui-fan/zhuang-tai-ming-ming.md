# 状态名

| 动词         | 简述  | 用途                     |
| ---------- | --- | ---------------------- |
| start      | 开始  | 开始(强调开始某个流程)           |
| pending    | 加载中 | 加载中                    |
| pause      | 暂停  | 暂停(强调流程的暂停，有可能后续会继续开启) |
| keep alive | 保持  | 保持某种状态                 |
| refresh    | 刷新  | 用于命名刷新(多用于页面,缓存等)      |
| stop       | 中止  | 停止(强调流程的终止)            |
| finish     | 完成  | 完成(强调流程的完成)            |
| close      | 关闭  | 关闭(多用于关闭状态,关闭文件等)      |
| destroy    | 销毁  | 销毁实例，常用语析构方法           |
| is         | 是否  | 是否是某个标准                |
